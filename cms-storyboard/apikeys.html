1. 한 줄 정의1. Định nghĩa một dòng

제휴몰 API 관리 = 제휴몰마다 발급받은 API 키(Access Key)의 상태와, 하루에 몇 번까지 호출할 수 있는지(일일 호출 한도)를 한곳에서 관제하는 화면. 포스트백이 "결제 끝났다"는 통보를 받는 창구라면, 이 화면은 그 통보를 받을 자격(키)이 살아있는지·막히지 않았는지를 지키는 관제탑이다. 키가 만료되거나 막히면 포스트백 자체가 끊긴다.Quản lý API đối tác = màn quản lý tập trung trạng thái key API (Access Key) được cấp cho từng cửa hàng liên kết, và giới hạn số lần gọi API mỗi ngày. Nếu postback là cửa nhận thông báo "đã thanh toán xong", thì màn này là đài kiểm soát canh xem tư cách nhận thông báo đó (key) còn sống hay bị chặn. Nếu key hết hạn hoặc bị chặn, bản thân postback cũng bị đứt theo.

API 키는 항상 마스킹 처리된다. 키가 노출되면 즉시 재발급 후 교체해야 하며, 키 교체 시 해당 제휴몰의 적립 추적이 일시 중단된다는 점을 운영자가 미리 알고 있어야 한다.Key API luôn được che (mask). Nếu key bị lộ phải tái cấp và thay ngay, và Vận hành viên cần biết trước rằng khi thay key, việc theo dõi tích lũy của cửa hàng đó sẽ tạm dừng.

3. 지금 왜 전부 '미확인'인가 — 테스트 API의 정체3. Vì sao hiện tất cả đều 'Chưa xác nhận' — Bản chất API test

한국 운영 담당자가 개발용으로 공유한 테스트 API(api.ddaynet.com/cou/reco_link.php)는 쿠팡 파트너스 정식 API가 아니라, 한국 담당자 측 제휴 대행사(ddaynet)가 자체 발급한 링크 생성 API다. 대행사가 보유한 쿠팡 파트너스 계정으로 경유 링크·광고 소재를 만들어주고, click_id·adid 파라미터로 클릭을 추적하는 방식이다. 정식 쿠팡 파트너스 API는 앱 패키지네임이 확정된 후에야 신청·발급받을 수 있다 — 그래서 이 화면의 Access Key·상태 항목이 전부 비어 있는 것이 지금 시점에서는 정상이다.API test do người phụ trách vận hành Hàn Quốc chia sẻ để phát triển (api.ddaynet.com/cou/reco_link.php) không phải API Coupang Partners chính thức, mà là API tạo link do công ty đại lý liên kết (ddaynet) phía Hàn Quốc tự cấp. Đại lý dùng tài khoản Coupang Partners của mình để tạo link đi qua·creative quảng cáo, và theo dõi click bằng tham số click_id·adid. API Coupang Partners chính thức chỉ có thể đăng ký·cấp phát sau khi package name của app được chốt — nên việc Access Key·trạng thái của màn này đang trống hết là điều bình thường ở thời điểm hiện tại.

5. 데모의 범위5. Phạm vi demo

이 화면은 실제 API를 호출하지 않는다. 'API 상태'·'일일 호출 한도'는 운영자가 수동으로 입력·갱신하는 값일 뿐이고, 'API 호출 현황' 그래프도 실 로그 연동이 없다. 지금은 "쿠팡 등 5개 제휴몰의 키를 등록할 자리와 항목을 확정"하는 기획 기준 화면이며, 실제 감시 기능은 정식 API 발급 이후에 붙는다.Màn này không gọi API thực. 'Trạng thái API'·'Giới hạn gọi hàng ngày' chỉ là giá trị Vận hành viên tự nhập·cập nhật, biểu đồ 'Tình trạng gọi API' cũng chưa kết nối log thực. Hiện đây là màn chuẩn thiết kế để "chốt vị trí và các mục cần đăng ký key cho 5 cửa hàng liên kết như Coupang", chức năng giám sát thực sự sẽ được gắn vào sau khi API chính thức được cấp phát.
